# Real Estate Website

A responsive real estate website built using HTML, CSS, and Bootstrap.
This project showcases a modern layout for property listings, services, blog content, and customer testimonials.

## Features

* Responsive navigation bar
* Hero carousel section
* Popular property listings
* Services section
* Blog section
* Testimonials
* FAQ accordion
* Footer with contact and social links

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons
